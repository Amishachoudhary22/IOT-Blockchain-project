# STEPS
1.clone the repository into desktop

2. install required dependencies

```pip install requirements.txt```

3.instantiate Ganache-cli Private Network.

```ganache-cli```

4.run app

```py app.py```

5.head into browser and type ```<Your computer ip>/<pin number>/<action>/```
